Font-Icons for Compass

Font-icons for Compass allow for quick deployment of pictogram icon sets in any compass web project. Using font glyphs as icons offer a number of advantages over images, such as vector scaling, ease of implimentation, colorizing on the fly, css shadows, etc. If you're still not convinced, check out this demo by Chris Coyier http://css-tricks.com/examples/IconFont/

Installation

# Get the plugin: 
  $ gem install font-icons --pre

# Install the plugin:
  $ cd font-icons
  $ gem install font-icons

# Add the following lines to your compass configuration file:
  require 'font-icons'

# Adding a font to your library is achieved by running the following command within an existing compass project:
  $ compass install -r font-icons font-icons/FONTNAME

# The following paramaters are accepted:
  iconic, entypo

Usage

// Import font-icons compass extension
@import "font-icons";

// Add the font family
@include font-icon-family(entypo);

// Style your heart out
ul.entypo {
  list-style: none;
  list-style-image: none;
  li {
    @include font-icon-base(entypo); 
    &.entypo-music { @extend .entypo-music; }
  }
}

Future Plans

Right now, this project assist in deploying icon sets and provides simple mixins to assist getting your icons on your pages. In future releases, there will be options to make quick changes to them, such as color modifications, shadows, stroke, etc.
